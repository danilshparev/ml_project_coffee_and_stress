# Прогнозирование уровня стресса с использованием машинного обучения
## Цель проекта

Построить и сравнить модели машинного обучения для прогнозирования уровня стресса человека на основе демографических, поведенческих и физиологических данных.

## Что было реализовано

- Предобработка и подготовка данных;
- Отбор признаков и исключение утечки данных;
- Обучение моделей **Logistic Regression**, **KNN** и **Random Forest**;
- Сравнение моделей по основным метрикам качества;
- Анализ результатов и выбор наиболее эффективной модели.

## Результат

По итогам сравнения лучшей моделью оказался **Random Forest**, который показал наиболее высокое качество классификации и был выбран для практического применения.
